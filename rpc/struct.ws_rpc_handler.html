Struct liberdus_rpc::rpc::ws_rpc_handler

pub struct ws_rpc_handler;
Expand description

WebSocket entry point for handling JSON-RPC requests.

This handler establishes a WebSocket connection, processes incoming JSON-RPC messages, and sends responses back. It supports subscription management via UUID generation.

§Parameters

  • state: Shared state containing the backend interface.
  • ws: The WebSocket connection.

§Returns

A WebSocket response handler.
